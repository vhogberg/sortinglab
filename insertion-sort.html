Insertion sort

How it works

Insertion Sort builds a sorted list one item at a time, like sorting playing cards in your hand! Each new card is placed in the correct position relative to the already sorted part.

  • üîπ Take one element ‚Üí Compare with sorted ones ‚Üí Insert it in the right spot among the sorted elements ‚Üí Repeat until sorted.

  • üîπ Best case: Already sorted? ‚úÖ Only takes O(n) time!

  • üîπ Worst case: Needs to shift many elements, runs in O(n¬≤).

  • üîπ Why learn it? It's efficient for small lists and nearly sorted data.
